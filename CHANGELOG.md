## [1.0.0] - 2026-01-04

### 追加
- プロジェクト監視機能（Active/Idle/Stale ステータス分類）
- Git連携機能（ブランチ、変更状況、ホットネススコア）
- クイックアクション（Explorer、ターミナル、VS Code、AI CLI起動）
- フィルタリング・ソート機能
- Windows自動起動機能

### 技術スタック
- Electron + React + TypeScript
- Tailwind CSS
- Chokidar（ファイル監視）
